Cursor 연동

노트 파일을 직접 AI 에디터로 열어 문서를 개선하고, 파편화된 메모에서 비즈니스 코드를 생성하거나 로직을 정교화합니다.

🤖

Claude Code

최신 언어 모델의 추론 능력을 옵시디언 로컬 저장소에 직접 투입하여 대량의 문서를 요약하고 인사이트를 도출합니다.

🔗

MCP (Model Context)

에이전트 표준 프로토콜을 통해 옵시디언의 지식을 다른 비즈니스 API와 연결, 실시간 정보 업데이트와 실행 자동화를 구현합니다.
